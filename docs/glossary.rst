Glossary
========

.. glossary::


   Wall-clock time

      Elapsed time, as perceived by a human being.

      See also:

      - :term:`Elapsed time`.


   Elapsed time

      Time consumed by a process while it is scheduled by the operating system
      (e.g. :term:`Wall-clock time` minus the time where the process is
      interrupted/suspended).

      See also:

      - :term:`Wall-clock time`.
